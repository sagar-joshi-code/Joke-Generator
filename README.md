# 😂 Joke Generator App

A simple web app that fetches and displays random jokes using a public API.

---

## 🚀 Features

- Fetches random jokes from an API
- Displays **setup** and **punchline**
- Shows loading text while fetching joke
- Clean and simple user interface
- Handles errors gracefully

---

## 🛠️ Technologies Used

- HTML
- CSS
- JavaScript (ES6)
- Fetch API

---

## 📖 How to Use

1. Open `index.html` in your browser
2. Click the **"Get Joke"** button
3. Wait for the setup
4. Punchline appears after a short delay 😄

---

## 🌐 API Used

- Official Joke API  
  https://official-joke-api.appspot.com/random_joke
